# 📊 Survey-Based PCA Analysis of Productivity & Success Factors

## 🧠 Problem
Understanding the factors that influence productivity and success is complex, as they depend on multiple psychological, social, and behavioral variables.

However:

- these factors are often subjective and multidimensional  
- raw survey data is difficult to interpret directly  
- correlations between variables are not always clear  
- there is a lack of structured insights from self-reported data  

This makes it challenging to extract meaningful patterns and actionable conclusions.

---

## 🎯 Solution
Design and execution of a data analysis pipeline based on a social media survey, followed by **Principal Component Analysis (PCA)** to uncover hidden structures in the data.

The system enables:

- transformation of raw survey responses into structured data  
- dimensionality reduction using PCA  
- identification of key latent factors  
- interpretation of relationships between variables  

---

## ⚙️ Technologies
- Python  
- Pandas, NumPy  
- Matplotlib / Seaborn  
- Scikit-learn (PCA)  
- Jupyter Notebook  

---

## 🚀 Key Features

### 📋 Data Collection
- Online questionnaire distributed via social media  
- Collection of multi-dimensional responses  

### 🧹 Data Preprocessing
- Data cleaning and formatting  
- Encoding of qualitative variables  
- Handling missing values  

### 📉 PCA Analysis
- Dimensionality reduction  
- Extraction of principal components  
- Explained variance analysis  

### 📊 Data Visualization
- Scree plot (variance explained)  
- Correlation circle  
- Projection of individuals  

### 🔍 Interpretation
- Identification of main influencing factors  
- Understanding variable contributions  
- Insight generation from components  

---

## 🏗️ Methodology

The project follows a structured pipeline:

1. Survey design and distribution  
2. Data collection from respondents  
3. Data cleaning and preprocessing  
4. PCA computation  
5. Visualization of results  
6. Interpretation of principal components  

---

## 📊 Results
The PCA allowed the identification of the most important dimensions structuring the data, reducing complexity while preserving key information.

It highlighted the main factors influencing productivity and success, enabling a clearer interpretation of relationships between variables.
